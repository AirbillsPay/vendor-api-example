---

Before creating a transaction for products that have predefined plans or providers — internet data, betting, cable TV, electricity, and transport — call the appropriate listing endpoint to get valid `prodId` and `prodAmount` values. Using IDs or amounts that come directly from these responses ensures your `/transact` calls reference plans that are currently active and correctly priced.

<Note>
  All listing endpoints require the `secretkey` header and accept no query parameters.
</Note>

---

## GET /list/internet — List Internet Data Plans

Retrieve all available mobile internet and data plans offered through Airbills. Each plan object includes a `prodId` and `prodAmount` that you pass directly into your `/transact` request.

### Endpoint

```
GET /api/vendor/gateway/list/internet
```

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `secretkey` | ✓ | Your business secret key |

### Response

<ResponseField name="status" type="string">
  `"00"` on success. Any other value indicates an error.
</ResponseField>

<ResponseField name="data" type="array">
  Array of available internet data plans. Each object contains a `prodId` and `prodAmount` to use in your `/transact` request.
</ResponseField>

### Code Example

```typescript
const response = await fetch(`${BASE_URL}/list/internet`, {
  headers: { 'secretkey': process.env.SECRET_KEY },
});
const { data } = await response.json();
// data: [{ prodId: '...', prodAmount: 1000, ... }]
```

## GET /list/elect — List Electricity Providers

Retrieve all supported electricity distribution companies (DISCOs). Each provider object includes an `electId` that you pass to `POST /validate/elect` before making an electricity purchase.

### Endpoint

```
GET /api/vendor/gateway/list/elect
```

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `secretkey` | ✓ | Your business secret key |

### Response

<ResponseField name="status" type="string">
  `"00"` on success. Any other value indicates an error.
</ResponseField>

<ResponseField name="data" type="array">
  Array of electricity distribution companies. Each object contains an `electId` you pass to `POST /validate/elect` before purchasing electricity.
</ResponseField>

### Code Example

```typescript
const response = await fetch(`${BASE_URL}/list/elect`, {
  headers: { 'secretkey': process.env.SECRET_KEY },
});
const { data } = await response.json();
```

<Note>
  Use the `electId` from this response in `POST /validate/elect` to verify a customer's meter number before calling `/transact`.
</Note>
