Use these endpoints to retrieve past transactions — either all transactions associated with your business account, or a single transaction looked up by its ID. These are useful for building transaction dashboards, reconciling payments, and checking the current status of a specific bill payment.

## GET /transaction/all

Retrieves all transactions associated with your business account. No query parameters required.

### Endpoint

```
GET /api/vendor/gateway/transaction/all
```

### Request Headers

| Header | Value |
|--------|-------|
| `secretkey` | `your-business-secret-key` |

### Response

<ResponseField name="status" type="string">
  `"00"` on success. See [Error Codes](/api-reference/error-codes) for all possible values.
</ResponseField>

<ResponseField name="data" type="array">
  An array of transaction objects for your business account.

  <Expandable title="Base fields (all transaction types)">
    <ResponseField name="id" type="string">Unique transaction UUID.</ResponseField>
    <ResponseField name="signature" type="string">On-chain transaction signature.</ResponseField>
    <ResponseField name="txType" type="string">
      Bill type: `"airtime"`, `"data"`, `"electricity"`, `"cable"`, `"betting"`, or `"transport"`.
    </ResponseField>
    <ResponseField name="status" type="string">`"Successful"`, `"Pending"`, or `"Failed"`.</ResponseField>
    <ResponseField name="amount" type="number">Bill amount in NGN.</ResponseField>
    <ResponseField name="amountInToken" type="number">Amount paid in USDT/USDC.</ResponseField>
    <ResponseField name="token" type="string">`"USDT"` or `"USDC"`.</ResponseField>
    <ResponseField name="fee" type="number">Transaction fee.</ResponseField>
    <ResponseField name="pubKey" type="string">User's wallet public key.</ResponseField>
    <ResponseField name="create_at" type="string">ISO timestamp.</ResponseField>
  </Expandable>

  <Expandable title="Extra fields by txType">
    | `txType` | Extra Fields |
    |----------|-------------|
    | `airtime` | `phoneNumber`, `networkId` |
    | `data` | `phoneNumber`, `networkId`, `prodId` |
    | `electricity` | `electId`, `meterNo`, `metertoken`* |
    | `cable` | `smartCardNo`, `prodId` |
    | `betting` | `customerId`, `prodId` |
    | `transport` | `phoneNumber`, `prodId` |

    *`metertoken` is only present after a successful electricity payment.
  </Expandable>
</ResponseField>

### Example Response

```json
{
  "status": "00",
  "data": [
    {
      "id": "abc123",
      "signature": "5Kj9...",
      "txType": "airtime",
      "status": "Successful",
      "amount": 500,
      "amountInToken": 0.35,
      "token": "USDC",
      "fee": 0,
      "pubKey": "wallet_address",
      "phoneNumber": "08012345678",
      "networkId": "01",
      "create_at": "2026-06-02T10:00:00.000Z"
    }
  ]
}
```

### TypeScript Example

```typescript
const response = await fetch(`${BASE_URL}/transaction/all`, {
  headers: { 'secretkey': process.env.SECRET_KEY },
});

const { data } = await response.json();
```

## GET /transaction/get

Retrieves a single transaction by its ID.

### Endpoint

```
GET /api/vendor/gateway/transaction/get
```

### Request Headers

| Header | Value |
|--------|-------|
| `secretkey` | `your-business-secret-key` |

### Query Parameters

<ParamField query="id" type="string" required>
  The transaction ID to look up. This is the `id` value returned by `/transact` in `data.id`.
</ParamField>

### Response

<ResponseField name="status" type="string">
  `"00"` on success. See [Error Codes](/api-reference/error-codes) for all possible values.
</ResponseField>

<ResponseField name="data" type="object">
  A single transaction object. Contains all base fields plus the `txId` field, and any extra fields for the specific `txType`. See the field tables above.
</ResponseField>

### Example Response

```json
{
  "status": "00",
  "data": {
    "id": "abc123",
    "txId": "AIR123456",
    "txType": "airtime",
    "status": "Successful",
    "amount": 500,
    "amountInToken": 0.35,
    "token": "USDC",
    "fee": 0,
    "pubKey": "wallet_address",
    "phoneNumber": "08012345678",
    "networkId": "01",
    "create_at": "2026-06-02T10:00:00.000Z"
  }
}
```

### TypeScript Example

```typescript
const params = new URLSearchParams({ id: 'business-transaction-uuid' });

const response = await fetch(`${BASE_URL}/transaction/get?${params}`, {
  headers: { 'secretkey': process.env.SECRET_KEY },
});

const { data } = await response.json();
```
